# Sanaré — Dashboard de Inversión (offline)

## Qué incluye
- `index.html` con pestañas: Resumen, KPIs, Sensibilidad, Preguntas.
- Gráficos en SVG (sin librerías externas, funciona offline).
- Sliders de sensibilidad: Factor de ventas y % de marketing.
- Exportación: KPIs a CSV y checklist a TXT.
- Botón de imprimir/guardar PDF.

## Cómo abrir
1) Descomprime el ZIP.
2) Abre `sanare_inversion_dashboard/index.html` con Chrome.

## GitHub Pages
- Sube la carpeta `sanare_inversion_dashboard/` a tu repo.
- Activa Pages y apunta a la carpeta/branch donde está `index.html`.
